# 프로젝트 소개
> 여행을 다니며 꼭 필요한 것들을 알차게 구성했습니다!
>   1. 가계부를 통해 여행예산과 결산을 간편하게 설정하고 확인할 수 있습니다.
>   2. 내가 가고자하는 여행지를 갔다온 사람들이 남긴 커뮤니티 글들을보며 여행계획을 미리 만드는데 도움을 받을 수 있습니다.
>   3. 여행지에 관한 정보를 찾아보고 저장하는 스크랩기능이 있습니다.
